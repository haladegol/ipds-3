🛡️ Signature Registry

Sync ET Open Rules ~10k sigs
ⓘ What this does

Downloads the latest Emerging Threats Open ruleset and compiles it into your registry.
Adds real-world PCRE patterns covering malware, exploits, and C2 traffic.

Sync Talos / Snort Rules community
ⓘ What this does

Downloads the Snort Community Ruleset (Cisco Talos, GPLv2, free — no key needed) and the Talos IP block list.
Parsed into TALOS-SID signatures with CVSS scores and MITRE ATT&CK mapping.

Activate All Signatures
ⓘ What this does

Sets every signature in the registry to active enforcing state in one click.
Use this to restore the full protection profile after a bulk deactivation.

Deactivate All Signatures
ⓘ What this does

Pauses every rule without deleting them — the engine will not fire on any pattern.
Useful for testing or isolating a false-positive storm.

Reset All Hit Counts
ⓘ What this does

Zeros the hit counter on every signature, giving you a clean measurement baseline.
Helpful after syncing new rules or starting a new monitoring period.

Purge Unused Signatures
ⓘ What this does

Permanently removes every rule that has never matched any traffic (hit count = 0).
Reduces registry size and improves match-scan performance.

Open Signature Editor
ⓘ What this does

Opens the full signature management page to search, filter, add, and delete individual rules.
Deploy custom PCRE patterns with MITRE ATT&CK mapping.

🔧 Maintenance

Purge All Blocked IPs
ⓘ What this does

Removes every entry from the block list.
Use after a false-positive event or to reset the IP block state.

Clear All Attack Logs
ⓘ What this does

Permanently deletes all attack log records tied to your account.
Does not affect signatures or blocked IPs — only the detection event history.

Flush SIEM Signal Log
ⓘ What this does

Empties the ips_signals.json file that feeds the SIEM integration pipeline.
Resets the rolling signal buffer without affecting the database.

Run Health Check
ⓘ What this does

Verifies the database connection, index integrity, and core engine components.
Results are written to the System Log and flashed as a status message.

Apply Strict IPS Profile
ⓘ What this does

Enables all 13 IPS protection toggles at once (inline filtering, shunning, DDoS, geo-IP, etc.).
Equivalent to setting every capability to ON in the Settings page.
